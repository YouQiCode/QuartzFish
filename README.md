## Setting如何使用？

### 设置闹钟

语句（设置为{HH}:{SS}）

```
$ /clock
$ {HH}
$ {MM}
```

实例（设置为01:23aa）

```
$ /clock
$ 01
$ 23
```

### 关闭闹钟

语句+实例

```
$ offclock
```

### 开启闹钟

语句+实例

```
$ onclock
```
